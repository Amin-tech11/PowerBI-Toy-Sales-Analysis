# Project Overview

## Introduction

The **Toy Sales Business Intelligence Dashboard** is an end-to-end Business Intelligence project developed using **Microsoft Power BI**.

The primary objective of this project is to transform raw sales data into meaningful business insights through data modeling, data transformation, DAX calculations, and interactive dashboard design.

The report enables business users to monitor company performance, evaluate profitability, analyze product sales, compare store performance, and make data-driven decisions using a centralized reporting solution.

---

# Business Goal

The project was designed to answer important business questions, including:

- Which stores generate the highest revenue?
- Which products contribute the most to total sales?
- How profitable is the business?
- How are sales changing over time?
- Which product categories perform best?
- Which stores require additional business attention?

Instead of relying on multiple spreadsheets and manual reporting, the dashboard provides a single interactive environment for business analysis.

# Solution Architecture

The project follows the standard Microsoft Power BI development process.

```
CSV Files
      │
      ▼
Power Query (ETL)
      │
      ▼
Data Model (Star Schema)
      │
      ▼
DAX Measures
      │
      ▼
Interactive Dashboard
      │
      ▼
Business Insights
```
